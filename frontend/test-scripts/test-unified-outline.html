Unified Outline Test Instructions

  1. Open the development server at http://localhost:5174/locker-placement-figma
  2. Open the browser console (F12)
  3. Copy and paste the following test script into the console:
// Test script for unified outline functionality
function testUnifiedOutline() {
  console.log('=== Testing Unified Outline Functionality ===');
  
  // Get all locker elements
  const lockers = document.querySelectorAll('[data-locker-id]');
  console.log(`Found ${lockers.length} lockers on the page`);
  
  if (lockers.length < 2) {
    console.log('Need at least 2 lockers to test multi-selection');
    return;
  }
  
  // Clear any existing selection first
  document.body.click();
  
  setTimeout(() => {
    // Select first locker with Ctrl
    console.log('\n1. Selecting first locker with Ctrl...');
    const firstLocker = lockers[0];
    const firstEvent = new MouseEvent('click', {
      bubbles: true,
      cancelable: true,
      view: window,
      ctrlKey: true
    });
    firstLocker.dispatchEvent(firstEvent);
    
    setTimeout(() => {
      // Select second locker with Ctrl
      console.log('2. Adding second locker to selection...');
      const secondLocker = lockers[1];
      const secondEvent = new MouseEvent('click', {
        bubbles: true,
        cancelable: true,
        view: window,
        ctrlKey: true
      });
      secondLocker.dispatchEvent(secondEvent);
      
      setTimeout(() => {
        // Check results
        console.log('\n=== RESULTS ===');
        
        // Check for unified outline
        const unifiedOutlines = document.querySelectorAll('.unified-selection-outline');
        console.log(`Unified outlines found: ${unifiedOutlines.length}`);
        
        if (unifiedOutlines.length > 0) {
          console.log('✅ Unified outline is being rendered!');
          unifiedOutlines.forEach((outline, index) => {
            const x = outline.getAttribute('x');
            const y = outline.getAttribute('y');
            const width = outline.getAttribute('width');
            const height = outline.getAttribute('height');
            console.log(`  Outline ${index + 1}: x=${x}, y=${y}, width=${width}, height=${height}`);
          });
        } else {
          console.log('❌ No unified outline found');
        }
        
        // Check individual outlines
        const individualOutlines = document.querySelectorAll('.selection-outline');
        console.log(`Individual selection outlines visible: ${individualOutlines.length}`);
        
        // Log selected lockers info
        const selectedCount = document.querySelectorAll('.locker-selected').length;
        console.log(`Selected lockers: ${selectedCount}`);
        
      }, 500);
    }, 500);
  }, 500);
}

// Run the test
testUnifiedOutline();

What to look for:

  • When selecting adjacent lockers, there should be ONE unified outline around the group
  • Individual dotted outlines should be hidden for lockers in the unified group
  • The unified outline should properly encompass all selected adjacent lockers
  • Check the console for debug messages about adjacency detection

Manual Testing:

  • Try selecting two adjacent lockers with Ctrl+Click
  • The unified outline should appear at the correct position and size
  • Try selecting non-adjacent lockers - they should show individual outlines
