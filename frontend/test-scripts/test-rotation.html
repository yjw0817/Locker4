🔄 락커 드래그 기반 회전 시스템 테스트

📋 테스트 체크리스트

1. 기본 회전 기능

  1. 락커를 선택하여 회전 핸들이 표시되는지 확인
  2. 회전 핸들을 드래그하여 자유롭게 회전 가능한지 확인
  3. 회전 중 실시간으로 각도가 표시되는지 확인
예상 결과: 락커가 마우스 움직임에 따라 부드럽게 회전하며, 현재 각도가 실시간으로 표시됨

2. 스냅 기능

  1. 회전 핸들을 드래그하여 0°, 45°, 90° 등 주요 각도 근처로 이동
  2. 5도 범위 내에서 자동으로 스냅되는지 확인
  3. 스냅될 때 녹색 원이 표시되는지 확인
예상 결과: 주요 각도(0°, 45°, 90°, 135°, 180°, 225°, 270°, 315°)에서 자동 정렬되고 시각적 피드백 제공

3. 키보드 수정자

  1. Shift + 드래그: 15도 단위로 제한되는지 확인
  2. Alt + 드래그: 스냅 기능이 비활성화되는지 확인
  3. 일반 드래그: 5도 범위에서 스냅이 작동하는지 확인
예상 결과: 각 수정자 키에 따라 회전 동작이 다르게 작동

4. 시각적 피드백

  1. 회전 중 락커가 약간 투명해지는지 확인 (opacity: 0.8)
  2. 회전 핸들 커서가 grab → grabbing으로 변경되는지 확인
  3. 각도 표시 배경이 흰색 박스로 표시되는지 확인
  4. 스냅 시 녹색 원 애니메이션이 작동하는지 확인
예상 결과: 모든 시각적 피드백이 정상 작동

5. 데이터 저장

  1. 락커를 회전시킨 후 마우스를 놓기
  2. 콘솔에서 회전 저장 로그 확인
  3. 페이지 새로고침 후 회전 상태 유지 확인
예상 결과: 회전값이 서버에 저장되고 새로고침 후에도 유지됨

⌨️ 키보드 단축키

  • R - 회전 모드 힌트 (콘솔에 메시지 표시)
  • Shift + 드래그 - 15도 단위로 제한
  • Alt + 드래그 - 스냅 기능 비활성화
