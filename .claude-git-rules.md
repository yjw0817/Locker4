# Claude Git 자동화 규칙

## 필수 자동 실행 규칙

### 파일 작업 후 자동 Git 작업
1. **파일 생성/수정 후**: 즉시 `git add <file>` 실행
2. **모든 작업 완료 후**: 자동으로 `git commit` 실행
3. **파일 삭제 시**: `git rm <file>` 실행

### 커밋 메시지 규칙
- feat: 새로운 기능
- fix: 버그 수정  
- refactor: 리팩토링
- perf: 성능 개선
- docs: 문서 수정
- style: 코드 포맷팅
- test: 테스트 코드

### 예외 상황
- 사용자가 "커밋하지 마세요" 또는 "git 작업 금지"라고 명시한 경우만 제외

## 실행 우선순위
이 규칙은 다른 모든 규칙보다 우선합니다.
